# Expense Tracker (Java)

A console-based Expense Tracker application developed using Java to help users manage and monitor their daily expenses efficiently. The application allows users to add, remove, and view expenses while keeping track of spending limits.

---

## 🔹 Features

- Add new expenses with details (amount, category, date, description)
- Remove expenses using unique ID
- Display all recorded expenses
- Automatic expense ID generation
- Expense limit monitoring with alert notification
- Preloaded sample expenses for demonstration

---

## 🔹 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList (Dynamic Data Storage)
- Scanner (User Input Handling)

---

## 🔹 Application Flow

1. User sets an expense limit
2. System displays menu options:
   - Add expense
   - Remove expense
   - Display all expenses
   - Exit application
3. User interacts with the system through console input
4. The system:
   - Stores expenses dynamically
   - Assigns unique IDs
   - Checks if total exceeds the limit
5. Displays warning if spending exceeds limit

---

## 🔹 How to Run

1. Save the file as:

   ExpenseTracker.java

2. Compile the program:

   javac ExpenseTracker.java

3. Run the program:

   java ExpenseTracker

---

## 🔹 Sample Functionalities

- Add Expense → Stores new expense details  
- Remove Expense → Deletes using ID  
- Display Expenses → Shows all stored records
